collaborate_you_room

issueを新規作成するときに、youRoomの任意のroomへ投稿できるpluginです。 インストール プラグインのインストール

Redmineをインストールしたディレクトリで下記コマンドを実行

$ cd vendor/plugins $ git clone https://github.com/asktkhr/redmine_collaborate_you_room.git $ cd ../.. $ rake db:migrate:plugins

プラグイン設定

{Rails.root}/vendor/plugins/collaborate_you_room/lib/oauth_consumer.rbをエディタで開き、youRoomのconsumer_key,consumer_secretを設定
